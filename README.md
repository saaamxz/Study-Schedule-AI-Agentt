# Study Schedule AI Agent 📚

An intelligent agent built with LangChain and Streamlit to assist students in querying their academic schedules and exam dates.

## Key Features
- **Smart Querying:** Ask questions about your study plan for any day.
- **Data-Driven:** Powered by CSV datasets for schedules and exams.
- **AI Assistant:** Uses Google Gemini (via LangChain) to provide accurate answers.
